# 制作背景
服装に気を使わない人が急に服装に気を遣おうとする意識転換にハードルを感じ、誰にでも簡単に使えるコーディネーションアプリを作りました。

# 工夫した点
コーディネーション提案の最適化 <br >
服によって金額帯が異なる点やどのような選定基準が最適かを考慮しました。

# 今後の展望
簡単にかつ最適なコーディネーション提案をするための改良をしていきたい。<br>
- データベース量の増加（webスクレイピング）
- 詳細なカテゴリ分類（公式スタイルポストやスタイリングアプリの参照）
- 提案のポスト形式での保存機能
